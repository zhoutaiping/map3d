## 产品概述

在现有3D地图组件中实现大区分布功能的层级下钻交互。

## 核心功能

1. **大区分布全国地图**：点击"业务大区分布"按钮时，使用 `regionGroupContainer` 渲染全国地图，省份按五大区分组显示对应颜色，同时显示散点数据
2. **大区级下钻**：点击大区中的省份时，地图下钻到该大区包含的所有省份（在 `regionGroupContainer` 中继续显示），显示该大区内的散点数据
3. **省份级下钻**：在大区地图中点击单独省份时，才下钻到单独省份详情（使用 `regionMapContainer`）
4. **容器切换**：`regionGroupContainer` 替换原地图（隐藏 `mapContainer`）

## 交互流程

```
中国地图(原容器) → [点击大区分布] → 大区全国地图 → [点击省份] → 大区地图 → [点击省份] → 省份详情
```

## 技术栈

- 框架：Vue 3 + TypeScript
- 图表库：ECharts + echarts-gl (3D地图)
- 地图数据：GeoJSON

## 实现方案

### 核心架构

在现有三层容器架构基础上，增加大区地图层级：

```
mapContainer (中国地图) ← 默认显示
    ↓ 点击"业务大区分布"
regionGroupContainer (大区地图) ← 替换显示
    ↓ 点击大区中的省份
regionMapContainer (省份详情) ← 覆盖显示
```

### 导航状态管理

使用 `navigationStack` 扩展来追踪当前层级：

- `level: 'china'` - 中国地图
- `level: 'region-group'` - 大区全国地图
- `level: 'region'` - 大区省份地图
- `level: 'province'` / `level: 'city'` - 省份/市级地图

### 关键技术点

1. **地图实例管理**：新增 `regionGroupChartInstance` 管理 `regionGroupContainer` 的 ECharts 实例
2. **动态地图数据**：大区地图需要筛选出该大区包含的省份 GeoJSON 数据
3. **散点数据过滤**：根据当前大区过滤 `scatterData` 中对应的散点

## 实现细节

### 1. 新增响应式变量

```javascript
const regionGroupContainer = ref(null);
let regionGroupChartInstance = null;
const currentRegionGroup = ref(null);  // 当前选中大区
const regionGroupLevel = ref('china'); // 'china' | 'region' 
```

### 2. renderRegionGroupMap 函数

- 在 `regionGroupContainer` 中初始化 ECharts 实例
- 渲染全国地图，省份按 `REGION_GROUPS` 配置着色
- 添加散点数据图层
- 设置点击事件：点击省份 → 下钻到大区

### 3. renderRegionGroupDrillDown 函数

- 根据点击的省份找到所属大区
- 筛选该大区所有省份的 GeoJSON 数据
- 在 `regionGroupContainer` 中渲染大区地图
- 过滤并显示该大区的散点数据
- 设置点击事件：点击省份 → 下钻到 `regionMapContainer`

### 4. 返回逻辑

扩展 `goBack()` 函数支持新的导航层级：

- 从省份详情返回 → 大区地图
- 从大区地图返回 → 大区全国地图
- 从大区全国地图返回 → 中国地图
