# 省份地图数据下载脚本

## 说明

此脚本用于从阿里云 GeoDataV 下载中国各省份的地图 JSON 数据。

## 使用方法

```bash
# 安装依赖（首次运行）
npm install

# 下载省份地图数据
npm run download-maps
```

## 输出

下载的文件将保存在 `public/maps/` 目录下，文件名格式为 `{adcode}.json`，例如：
- `110000.json` - 北京市
- `310000.json` - 上海市
- `440000.json` - 广东省

## 数据来源

数据来源：阿里云 GeoDataV (https://geo.datav.aliyun.com/areas_v3/bound/)

## 注意事项

1. 台湾省 (710000) 的数据在 GeoDataV 上不可用，因此无法下载
2. 如果文件已存在，脚本会自动跳过，不会重复下载
3. 脚本使用并发下载（默认 5 个并发），以提高下载速度
4. 下载过程中会显示进度信息
