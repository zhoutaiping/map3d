# 2026-03-26

## scatter3D tooltip 优化

- 省份下钻 tooltip 不显示：给 `map3D` series 加 `tooltip: { show: false }`，防止地图区域事件拦截散点 tooltip
- hover 体验差：symbolSize 20→25，opacity 0.8→0.9，加 `blendMode: "source-over"`
- emphasis 状态：border 加粗至 3，hover 颜色变亮（光伏 #73d13d，风电 #ffa940）
- tooltip position 改为自适应偏移，根据鼠标位置判断左右上下，避免超出屏幕边界
- 同步更新 `createScatterTooltipConfig` 和 `createRegionGroupTooltipConfig` 两处
